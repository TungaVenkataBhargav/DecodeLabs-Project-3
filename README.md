# DecodeLabs-Project-3

ATM Interface System

Project Overview

The ATM Interface System is a Java-based console application that simulates the basic operations of an Automated Teller Machine (ATM). The project 

is developed using Object-Oriented Programming (OOP) principles and allows users to securely access their account using a PIN, perform 

transactions, and manage their balance.

-> Features :

Secure ATM PIN Authentication

Deposit Money

Withdraw Money

Check Account Balance

Input Validation

Insufficient Balance Handling

Menu-Driven Interface

Three PIN Attempt Security System

-> Technologies Used :

Java

OOP Concepts

Scanner Class

Conditional Statements

Loops

Methods & Classes

-> Workflow

1.User enters ATM PIN.

2.System validates the PIN.

3.User gets three attempts to enter the correct PIN.

4.After successful login, the ATM menu is displayed.

5.User can :

~ Deposit money

~ Withdraw money

~ Check account balance

~ Exit the system

6.The system validates every transaction and displays appropriate messages.
